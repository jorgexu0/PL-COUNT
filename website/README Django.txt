Instalar Bibliotecas necessárias

Linha de comandos(cmd): 		Para que serve?

-pip install django_filters		-> filtros dos objetos
-pip install psycopg2 			-> postgresql
-pip install django_crispy_forms 	-> alteração dos perfils


caso base de dados dê erro:
-pip install djongo		


Para correr o servidor django é necessário estar na diretoria /django/app/:

Na Linha de comandos(cmd):		Para que serve?
-python manage.py runserver		-> Para ligar o servidor


-python manage.py createsuperuser	->criar um administrador 


Fazer este passo quando estiver a correr o django pela primeira vez

Na Linha de comandos(cmd):		Para que serve?
-python manage.py makemigrations 	->alterações na base dados
e	
-python manage.py migrate		-> Migrar
